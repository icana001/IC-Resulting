## Export als PDF

### Option 1: Browser Print-to-PDF

1. Öffne `onepager-a4.html` im Browser (Chrome/Edge empfohlen)
2. Drücke `Strg + P` (Windows) oder `Cmd + P` (Mac)
3. Wähle **"Als PDF speichern"** als Drucker
4. Einstellungen:
   - Papierformat: A4
   - Ränder: Standard oder Keine
   - Hintergrundgrafiken: ✓ Aktivieren
5. Speichern unter: `/marketing/exports/ki-automatisierung/onepager-a4.pdf`

### Option 2: SVG Export

1. Öffne `onepager-a4.svg` in einem Vektorprogramm
2. Exportiere als PDF
